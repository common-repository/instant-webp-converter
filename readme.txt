Instant WebP Converter automatically converts JPEG and PNG images to WebP format to enhance website performance and speed. 

### Watch Tutorial:
Here is a detailed video guide to help you understand the plugin: [Watch Tutorial](https://youtube.com/live/aG5toHp9vmQ).

== Installation ==

1. Upload the `instant-webp-converter` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. The plugin will automatically convert existing images on activation.

== Frequently Asked Questions ==

= Will the original images be deleted? =
Yes, the plugin deletes original JPEG and PNG images after converting them to WebP format to save storage space.

= Does this plugin support SVG images? =
Currently, the plugin only supports JPEG and PNG image formats.
